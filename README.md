# Cookiecutter My Project Starter

A custom **Cookiecutter template** designed to quickly generate a clean, organized, and production-ready project structure for data science projects.

---

## Features

- Well-organized project structure  
- Auto-generated folders for data, notebooks, scripts, and reports  
- Easily customizable project name and author details  
- Ready to integrate with GitHub  
- Ideal for data science.
